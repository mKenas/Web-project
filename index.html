Features

Screen sharing

To aid the user in the most optimal way, we have a screen sharing alternative, where the IT-guide can access the screen of the user and aid the in a quick and easy way. The app will hide all sensitive information from the IT-guide, security is on top of our priority list.

Number two

Chat and video

Having both chat and video sharing allows the user to choose how they want to receive the IT support, thus making the app user-oriented.

Considering problems with language barriers, the app will include multilingual guides, as well as guides who speak other languages than english. This grants optimal IT support in the language of the user's preference. The universal design of this app allows users with disabilities to receive IT support as well.

Guide evaluation

The guide evaluation system allows the user to rate the help provided by the IT guide. This rating is visible for other uses, and it allows the user to select the IT guide of their preference, based on their rating score.

Share your screen seemlessly and get concrete help when ever you need it.
